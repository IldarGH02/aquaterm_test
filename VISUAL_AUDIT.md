## MODAL TRAP FOCUS ПРОБЛЕМЫ

### ❌ ТЕКУЩЕЕ:

```
Нажимаем Tab в модальном окне:
───────────────────────────────

[Modal]
  ├─ [Name Input] → фокус
  ├─ [Phone Input]
  ├─ [Submit Button]
  ├─ [Close Button]
  │
  └─ [ВНЕЗАПНО Tab уходит ПОЗАДИ модали!] ❌
      │
      └─ [Фокус на Header Link] (позади блюра)
      
Result: Запутанная навигация! Нарушение WCAG AAA
```

### ✅ ПРАВИЛЬНОЕ (FOCUS TRAP):

```
Нажимаем Tab в модальном окне:
───────────────────────────────

[Modal]
  ├─ [Name Input] → фокус
  │    ↓ Tab
  ├─ [Phone Input]
  │    ↓ Tab
  ├─ [Service Select]
  │    ↓ Tab
  ├─ [Submit Button]
  │    ↓ Tab
  ├─ [Close Button]
  │    ↓ Tab (ЗАЦИКЛЕНО - вернулось на первый элемент)
  └─ [Name Input] ✅

Result: Фокус "заперт" внутри модали, как и должно быть!

Реализация:
useEffect(() => {
  const modal = modalRef.current;
  if (!modal) return;
  
  const focusableElements = modal.querySelectorAll(
    'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])'
  );
  const firstElement = focusableElements[0];
  const lastElement = focusableElements[focusableElements.length - 1];
  
  const handleKeyDown = (e) => {
    if (e.key === 'Tab') {
      if (e.shiftKey) {
        if (document.activeElement === firstElement) {
          e.preventDefault();
          lastElement.focus();
        }
      } else {
        if (document.activeElement === lastElement) {
          e.preventDefault();
          firstElement.focus();
        }
      }
    }
  };
  
  modal.addEventListener('keydown', handleKeyDown);
  return () => modal.removeEventListener('keydown', handleKeyDown);
}, []);
```

## ACCESSIBILITY (a11y) PROBLEMS

### ❌ ТЕКУЩЕЕ (NO ARIA):

```html
<!-- Screen reader читает: "Кнопка" (больше не знает!) -->
<button onClick={onCtaClick}>
  Вызвать инженера
</button>

<!-- Screen reader читает: "Меню (не понятно что это)" -->
<button className="lg:hidden p-2" onClick={() => setIsMenuOpen(!isMenuOpen)}>
  {isMenuOpen ? <X /> : <Menu />}
</button>

<!-- Screen reader: "Ссылка" (не понятно куда идет) -->
<a href="#services">Услуги</a>

<!-- Screen reader: "Изображение" (не понятно что изображено) -->
<img src="/1.png" />
```

### ✅ ПРАВИЛЬНОЕ (WITH ARIA):

```html
<!-- Screen reader: "Кнопка: Вызвать инженера" -->
<button 
  onClick={onCtaClick}
  aria-label="Вызвать инженера для консультации"
>
  Вызвать инженера
</button>

<!-- Screen reader: "Кнопка переключение меню навигации, нажато" -->
<button 
  className="lg:hidden p-3 min-w-[48px] min-h-[48px]"
  onClick={() => setIsMenuOpen(!isMenuOpen)}
  aria-label={isMenuOpen ? "Закрыть меню" : "Открыть меню"}
  aria-expanded={isMenuOpen}
  aria-controls="mobile-nav"
>
  {isMenuOpen ? <X /> : <Menu />}
</button>

<!-- Screen reader: "Ссылка: Услуги (в заголовке)" -->
<a href="#services" aria-label="Перейти к разделу Услуги">Услуги</a>

<!-- Screen reader: "Изображение: Установка системы отопления в коттедже" -->
<img 
  src="/1.png"
  alt="Установка системы отопления в коттедже площадью 220 м²"
/>
```
